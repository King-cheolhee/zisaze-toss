## 콘솔 붙여넣기용 (아래 전체)

```
[서비스 소개]
지사제는 정부·지자체·공공기관의 중소기업/소상공인 지원사업 공고를 한곳에 모아
검색하고, 사용자가 고른 선호 분야에 맞춰 추천해 드리는 정보 서비스예요.

검수에 참고해 주세요:

1. 외부 이동(외부 링크)에 대해
- 상세 화면의 [원문 보기]는 정부·공공기관이 게시한 공고 원문을 확인하는 용도예요.
  (허용 예외인 "공공기관 공식 페이지 연결", "단순 정보 확인용 이동"에 해당해요.)
- 그 외 외부 이동은 개인정보처리방침·이용약관(법적 고지) 2건뿐이에요.
- 모든 외부 이동은 토스 SDK의 openURL로만 처리했어요. 핵심 기능(검색·추천·AI 요약·북마크)은
  전부 앱 안에서 완결돼요.

2. 생성형 AI 고지에 대해
- 상세 화면의 요약 영역에 "AI 요약" 라벨과 "원문과 다를 수 있어요" 한계 고지를 함께 표시해요.
- 온보딩·설정의 선호 입력 화면에 "입력하신 선호는 AI가 분석해 맞춤 추천에 활용해요" 고지를
  인라인 문구로 표시해요(팝업 아님).
- 홈 추천 영역에는 "회원님의 선호 기반 추천" 라벨을 상시 표시해요.

3. 로그인·사용자 식별에 대해
- 자체 로그인·회원가입 화면이 없어요. 토스가 제공하는 익명 식별키(getAnonymousKey)로만
  사용자를 구분해요.
- 수집 정보는 익명 식별키, 선호 분야(선택), 사업장 지역(선택), 북마크가 전부이며
  이름·연락처 등 개인 식별 정보는 수집하지 않아요.

4. 온보딩에 대해
- 온보딩 두 단계(선호 분야, 지역) 모두 "건너뛰기"를 제공하고, 필수 입력이 하나도 없어요.
- 진입 즉시 뜨는 팝업·바텀시트는 없어요.

5. 데이터 삭제에 대해
- 설정 > "내 데이터 삭제"에서 확인 후 즉시 서버 데이터(선호·지역·북마크)를 삭제하고
  새 사용자로 시작해요.

6. 데이터 출처(네트워크 호출)에 대해
- 지원사업 데이터는 자사 운영 API 서버(https://www.zisaze.com/api)에서 HTTPS로만 받아요.
- 미니앱 안에 자사 웹사이트·다른 서비스로 유도하는 링크나 문구는 없어요
  (위 1번의 법적 고지 2건 제외).

문의: home143@naver.com
```

## 심사 Q&A 대비 (반려·질의 시 답변 근거 — 콘솔에는 넣지 않음)

| 예상 질의 | 답변 근거 |
|---|---|
| "원문 보기가 외부 의존 아닌가?" | 목록·검색·추천·AI 요약·북마크는 전부 앱 내 완결. 원문은 공공기관 공고 확인용(허용 예외). openURL만 사용, `src/lib/bridge.ts` 스킴 검증(http/https만) 포함 |
| "AI 표시 안 보이는 화면이 있다" | AI 산출물이 있는 곳 전부에 라벨: 상세(AI 요약 배지+한계 고지), 온보딩·설정(분석 고지), 추천(선호 기반 라벨). 감사 H16 pass(조건부 렌더 탈락 없음 확인) |
| "로그인 없이 어떻게 데이터가 유지되나?" | getAnonymousKey 해시 ↔ 서버 사용자 매핑. 북마크·선호는 서버 저장이라 재설치에도 유지(hash 영속 전제 — P3-3 실기기 검증 항목) |
| "왜 만 19세 이상인가?" | 사업자(중소기업·소상공인) 대상 정보 서비스 |
| "다크모드가 안 된다" | 앱인토스 규격상 라이트 고정(H4). color-scheme light 메타 + 다크 분기 코드 없음 |
| "번들에 eval이 있나?" | 소스·빌드 산출물 grep 0건(H2 게이트, P2-8·P4 재검증) |

## 콘솔 앱 설명란 주의 (P0-2 등록 시)

- **MVP에 실제 있는 기능만** 기술: 지원사업 검색·목록 / 선호 기반 맞춤 추천 / AI 요약 표시 / 원문 확인 / 북마크 / 선호·지역 설정 / 데이터 삭제.
- 웹에만 있는 기능(AI 상담, AI 탐색, 알림, 배너)은 **절대 언급 금지** — "설명에 있는데 앱에 없으면" 반려 사유.
